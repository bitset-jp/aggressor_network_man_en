Battery Replacement of RTC
^^^^^^^^^^^^^^^^^^^^^^^^^^^^

.. note::

   - Use ``CR1225`` battery
   - 交換は電源を切った状態で行って下さい。
   - Turn off the power before replacing the battery.


**交換手順:**

#. Slide open the lid of the main body.

#. Replace the battery.

   .. figure:: /_static/rtc.png
      :width: 50%

      RTC

#. After Replacement, execute :ref:`rtc set command<rtc_command>` to set RTC time.
