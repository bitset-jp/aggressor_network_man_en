5.1. Additional Information¶

5.1.1. Battery Replacement of RTC¶

Note

  • Use CR1225 battery

  • 交換は電源を切った状態で行って下さい。

  • Turn off the power before replacing the battery.

交換手順:

  1. Slide open the lid of the main body.

  2. Replace the battery.

  3. After Replacement, execute rtc set command to set RTC time.
