4.2. General Commands¶

4.2.1. ifconfig: Display network settings¶

Displays the settings of the selected network device (eth0, eth1).

Table 4.6 Devices displayed by ifconfig¶

Cursor

Display content

0 >

Display only eth0

1 >

Display only eth1

a >

Display both eth0 and eth1

Format:

> ifconfig

Execution example:

4.2.2. clear: Clear the main screen¶

Clears the main screen(0:main screen).

Format:

> clear

4.2.3. help: Display help¶

Displays the help screen on “1:help screen”.

Format:

> help {command_name}

Execution example:

4.2.4. stat: Redisplay the traffic statistics screen¶

Redisplay the traffic statistics screen on “2:stat screen”.

Format:

> stat

Note

If there was no active network device when booting, activating the network device will not display the traffic statistics screen. In that case, execute “stat” command.

4.2.5. iftop¶

Make “3:eth0 screen” and “4:eth1 screen” to display iftop.

Selected network device(s) will change to iftop mode.

Format:

> iftop

4.2.6. tcpdump¶

Make “3:eth0 screen” and “4:eth1 screen” to display tcpdump.

Selected network device(s) will change to tcpdump mode.

Format:

> tcpdump

4.2.7. height¶

Change the height of “3:eth0 screen” and “4:eth1 screen”.

Format:

> height {number_of_lines}
Table 4.7 arguments of height command¶

Argument

Required

Value

number_of_lines

Yes

10 - 35

Execution example:

> height 20
